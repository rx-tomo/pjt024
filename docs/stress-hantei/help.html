基本の使い方と用語（1ページ）
アプリに戻る GitHub
使い方（最短）
  1. 「データ選択」：デモデータを選ぶ（必要ならCSV読み込み）
  2. 「分析選択」：判定図/ランキング/分布/シミュレーションを選ぶ
  3. 「分析結果」：性別タブ・集計・フィルタで比較しながら確認
用語（やさしめ）
  • 4尺度：簡易版12項目（q1..q12）から作る4つの集計点（量的負担/コントロール/上司支援/同僚支援）。
  • 健康リスク：係数と式（100×exp(...)）から算出する相対指標。標準集団の平均を100として表します。
  • 総合健康リスク：図1（量的負担×コントロール）と図2（上司支援×同僚支援）の健康リスクを合成した値（図1×図2÷100）。
  • 全国平均: legacy：テクニカルノート掲載の全国平均値（デフォルト）。
  • 全国平均: updated：テクニカルノート中で併記されている訂正値（あるいは更新値）を使うモード。
  • 画像依存せず描画：判定図の等リスク線は、元サイトの画像を貼り付けるのではなく、係数と式からブラウザで計算して線を引いています。
  • 集計（groupBy）：部署/職種/拠点/勤務形態/年代など、どの切り口で平均・リスクを出すか。
  • フィルタ：対象を絞って集計する（例：拠点=東京だけ、など）。
指標の説明（図1/図2/総合健康リスク）

本アプリは、公開資料（入口ページ配下のテクニカルノート／結果の読み方）の説明に基づき、ブラウザ内で係数と式を用いて計算します。

  • 4尺度（q1..q12）：回答（1〜4点）を合計して尺度点にします。
    量的負担=q1+q2+q3 / コントロール=q4+q5+q6 / 上司支援=q7+q9+q11 / 同僚支援=q8+q10+q12
  • 図1の健康リスク：量的負担×コントロールの組み合わせから算出します（全国平均からのずれを指数関数で換算）。
    risk1 = 100 × exp((workload - A)×α + (control - B)×β)
  • 図2の健康リスク：上司支援×同僚支援の組み合わせから算出します。
    risk2 = 100 × exp((supervisorSupport - C)×γ + (coworkerSupport - D)×δ)
  • 総合健康リスク：図1と図2のリスクを合成した指標です。
    total = risk1 × risk2 ÷ 100
  • 全国平均（A,B,C,D）：基準となる平均値です。legacy（デフォルト）/ updated を切り替えられます。
  • 係数（α,β,γ,δ）：テクニカルノートに示されている係数です（性別で異なるため、画面は男女別タブで表示します）。
  • この数値が表すもの：全国平均を100とした「相対指標」です。100より大きいほど全国平均より高い傾向、100より小さいほど低い傾向として読みます（厳密な読み方は元資料に従います）。
注意（デモの位置づけ）
  • 表示しているデモデータはダミーです（実データは含みません）。
  • シミュレーションは「式に入れたらどう変わるか」の試算であり、施策の因果効果を保証するものではありません。
  • 医療判断・診断のためのツールではありません。
